Teooria

HTML tähendus.

HTML on keel, milles märgendatakse veebilehti.

HTML loob struktuuri (skeleti), CSS annab välimuse. Standardid tagavad, et kood töötab kõikjal.

Veebileht koosneb alati kahest suuremast osast: päisest (<.head>) ja sisust (<.body>). Kõige ülal on deklaratsioon , mis ütleb brauserile, et dokument järgib HTML5 standardit. See väldib nn quirks-mode’i, kus brauserid püüavad vana koodi "arvata". Koodi kirjutades ei lisata punkti peale linnukesi.

Miks peab style kaust linkima? Kui stiilid on eraldi failis, siis:

  1. Kood on puhtam – HTML sisaldab vaid sisu ja struktuuri, mitte kujundust.
  2. Taaskasutus – sama CSS fail saab kujundada mitut HTML lehte.
  3. Jõudlus – brauserid saavad CSS-faili vahemällu salvestada ja kiiremini laadida.
  4. Hooldatavus – stiile saab muuta ühes kohas ja need rakenduvad kõikjal.
